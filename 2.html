2. O que são seletores CSS? Quais são os tipos existentes? Seletores CSS são padrões usados para "selecionar" elementos HTML específicos para aplicar as regras de estilo. Eles são essenciais para direcionar regras específicas a elementos específicos dentro da estrutura HTML de uma página. Seletor de Tipo: seleciona todos os elementos de um tipo específico. Seletor de Classe: seleciona elementos que possuem uma classe específica. Seletor de ID: seleciona um elemento com um ID específico. Seletor Universal: aplica estilos a todos os elementos. Seletor de Atributo: seleciona elementos com atributos específicos. Seletores Combinados: como seletores de descendente aplica estilo a elementos filhos de outros, adjacente para elementos irmãos, e outros.
