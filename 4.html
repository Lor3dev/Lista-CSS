4.Como podemos incluir CSS em documentos HTML? CSS Inline: Aplicado diretamente no elemento HTML, usando o atributo style. Esse método é útil para estilizar um único elemento de forma rápida, mas não é ideal para estilos complexos. CSS Interno (Embedded): Colocado dentro de uma tag style no cabeçalho do documento HTML. Esse método permite definir estilos que se aplicam a várias partes da página, mantendo o CSS no mesmo arquivo HTML. CSS Externo: Armazenado em um arquivo separado (.css), vinculado ao documento HTML com uma tag no cabeçalho. Esse é o método mais recomendado para projetos maiores, pois permite a reutilização de estilos em várias páginas e facilita a manutenção.
