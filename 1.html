1. O que é uma regra CSS? Uma regra CSS (Cascading Style Sheets) é um conjunto de instruções que define o estilo de elementos HTML em uma página web. Cada regra CSS é composta por um seletor que identifica os elementos aos quais o estilo se aplica, uma propriedade que define oque o seletor vai fazer e um valor que especifica o valor dessa propriedade. As regras CSS são usadas para personalizar a aparência de uma página web, como cores, margens e fontes, com a intenção a torná-la visualmente mais atraente e funcional. Exemplo de regra CSS: div{ color: blue; font-size: 16px; }
